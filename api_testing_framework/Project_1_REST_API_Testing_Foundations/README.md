## Learning Path

### Phase 1: Foundations (Lessons 1.1-1.4)
Learn the theory behind API testing, REST principles, HTTP methods, and status codes.

### Phase 2: Tools Setup (Lessons 1.5-1.7)
Set up your development environment with pytest and httpx.

### Phase 3: Hands-On Testing (Lessons 1.8-1.13)
Write real API tests for all HTTP methods with httpx.

### Phase 4: Advanced Topics (Lessons 1.14-1.18)
Master response validation, JSON parsing, headers, parameters, and timeouts.

### Phase 5: Best Practices (Lesson 1.19)
Learn industry best practices for maintainable, scalable API tests.

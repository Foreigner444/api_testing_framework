# Lesson 5.18: JSON Configuration Files

## A. Concept Overview

### What & Why
**JSON configuration** provides machine-readable, structured configuration commonly used in web applications and APIs. Important when integrating with systems that produce JSON config or when configuration needs to be programmatically generated.

## B. Code Implementation

```python
import json
from pathlib import Path
from pydantic_settings import BaseSettings


class Settings(BaseSettings):
    api_key: str
    database_url: str
    timeout: int
    
    @classmethod
    def from_json(cls, path: Path):
        with open(path) as f:
            data = json.load(f)
        return cls(**data)


# config.json:
# {
#   "api_key": "test_key",
#   "database_url": "postgresql://localhost/db",
#   "timeout": 30
# }

settings = Settings.from_json(Path("config.json"))
```
